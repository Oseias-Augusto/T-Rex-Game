<h1 align="center">Núcleo de Prática de Programação (N.P.P)</h1>

Este é um projeto surgido da ideia de treinar as habilidades de programação e aprender a usar novas linguagens bibliotecas e frameworks.
Os projetos são decididos através de sorteios de temas e linguagens e podem durar uma semana, para projetos simples, ou mais, para projetos mais complexos.
